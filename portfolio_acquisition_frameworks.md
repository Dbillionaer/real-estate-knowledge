# 🏢 Portfolio Acquisition Frameworks

## Overview
This document provides detailed frameworks for acquiring multiple properties from single sellers or portfolios. These strategies are based on real conversations and complement the existing playbooks by showing how to scale from single-property deals to portfolio-level acquisitions.

## 📊 **PORTFOLIO ACQUISITION STRATEGIES**

### **Strategy 1: Multi-Property Portfolio Approach**
**From: `Subject To Pitch 7 - Portfolio of Properties.txt`**

**Portfolio Details:**
- **Total Properties**: 26 rental units
- **Blanket Mortgage**: $2.8M covering all properties
- **Property Types**: Mix of single-family and townhouses
- **Market Areas**: Philadelphia metropolitan area

**Framework Structure:**
```
Phase 1: Portfolio Assessment
├── Identify blanket mortgage structure
├── Assess per-property allocation ($107.7k average)
├── Evaluate total portfolio value vs. individual sales
└── Determine acquisition capacity (5 properties initially)

Phase 2: Per-Property Analysis
├── Individual property values ($100k-$200k range)
├── Rental income per property ($1,200/month average)
├── Mortgage allocation per property ($107.7k)
└── Equity position per property ($50k average)

Phase 3: Network Integration
├── Identify other interested buyers
├── Coordinate with seller's agent network
├── Structure phased acquisition approach
└── Maintain seller relationship for future deals

Phase 4: Funding Structure
├── Partner integration for larger acquisitions
├── Per-property financing allocation
├── Commission structure across portfolio
└── Legal structure for multi-property transfer
```

**Advanced Techniques:**
1. **Blanket Mortgage Navigation**: Understanding portfolio-level financing
2. **Per-Property Allocation**: Fair distribution of debt across properties
3. **Network Leverage**: Connecting seller with additional buyers
4. **Phased Acquisition**: Starting with 5 properties, expanding to full portfolio

**Strategy**: Phased acquisition with network integration

1. **Portfolio Assessment**: Identify blanket mortgage structure and allocation
