# 👥 Buyer Profiles

> This directory stores detailed buyer profiles written by the Dispositions Manager agent. Each buyer gets their own file for rich searchability.

## File Naming Convention

```
[buyer-name-slug].md
```

Example: `john-smith-huntsville-cash-buyer.md`

## Buyer Profile Template

```markdown
---
```
